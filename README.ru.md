Браузерное расширение на Manifest V3, которое делает веб-приложение **Claude** (`claude.ai` /
`claude.com`) справа налево и позволяет полностью настроить его типографику и тему.

![icon](icons/icon128.png)

> **Независимое, неофициальное расширение — не связано с Anthropic, не одобрено и не
> спонсируется ею. «Claude» является товарным знаком Anthropic.**

---

## ✨ Возможности

- **Письмо справа налево и направление** — `Auto` (определение по каждому блоку,
  лучше всего подходит для двуязычных чатов), принудительный RTL или принудительный LTR.
  Применяется по отдельности к ответам Claude, вашим сообщениям, полю ввода и списку
  чатов в боковой панели.
- **Выравнивание** — по началу строки (следует направлению), по правому краю, по центру,
  по левому краю или по ширине.
- **50+ шрифтов, сгруппированных по письменности** — персидская, арабская, иврит, урду,
  латиница, кириллица, деванагари, японская, китайская, корейская, тайская, бенгальская —
  с поиском, предпросмотром в выборе и фильтрами по языкам. Отдельные шрифты для
  основного текста, заголовков и кода — каждый является загружаемым веб-шрифтом
  (Google Fonts / jsDelivr).
- **Типографика** — размер шрифта, насыщенность, высота строки, межбуквенный, межсловный
  и межабзацный интервалы, а также ограничение ширины для чтения — всё это с ползунками
  в реальном времени.
- **Локальные цифры** — показывайте числа цифрами вашего языка (персидскими, арабскими,
  урду, пушту, хинди, бенгальскими, тайскими). Заблокировано с пояснением для языков,
  которые уже используют стандартные западные цифры.
- **Математика и код остаются корректными** — формулы KaTeX и блоки кода сохраняют
  направление слева направо и остаются нетронутыми.
- **Темы Claude** — переключайте сам Claude между Auto / Светлой / Тёмной / контрастной
  Чёрной, а также задавайте собственный акцентный цвет.
- **Интерфейс на 25 языках** — панель настроек при первом запуске автоматически определяет
  язык вашего браузера и полностью переведена на 25 языков.
- **Предпросмотр в реальном времени** — увидьте, как именно будет выглядеть Claude
  (его собственная тема + акцент) перед применением.
- **Пресеты, экспорт/импорт, пользовательский CSS** и отшлифованный, доступный с клавиатуры
  интерфейс со светлой / тёмной / чёрной / системной темами панели.

## 🚀 Установка (режим разработчика)

1. Откройте `chrome://extensions` (Chrome, Edge или Brave).
2. Включите **Режим разработчика** (в правом верхнем углу).
3. Нажмите **Загрузить распакованное расширение** и выберите папку этого проекта.
4. Откройте чат Claude и нажмите на значок расширения, чтобы открыть настройки.

Сочетания клавиш можно назначить на странице `chrome://extensions/shortcuts` (по умолчанию
ни одно не задано).

## 🔒 Конфиденциальность

Расширение **не собирает никаких данных** — никакой аналитики, телеметрии или отслеживания.
Настройки хранятся только в `chrome.storage.local` на вашем устройстве. Единственные
сетевые запросы — это **файлы шрифтов**, загружаемые с Google Fonts / jsDelivr, когда вы
выбираете загружаемый шрифт. Подробности см. в [PRIVACY.md](PRIVACY.md).
