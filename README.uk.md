Розширення для браузера на базі Manifest V3, яке робить вебзастосунок **Claude** (`claude.ai` /
`claude.com`) справа наліво та дозволяє повністю налаштувати його типографіку й тему.

![icon](icons/icon128.png)

> **Незалежне, неофіційне розширення — не пов’язане з Anthropic, не схвалене та
> не спонсороване нею. «Claude» є торговою маркою Anthropic.**

---

## ✨ Можливості

- **Справа наліво та напрямок** — `Auto` (визначення для кожного блоку, найкраще для
  двомовних чатів), примусовий RTL або примусовий LTR. Застосовується незалежно до
  відповідей Claude, ваших повідомлень, поля введення та списку чатів у бічній панелі.
- **Вирівнювання** — за початком (відповідає напрямку), праворуч, по центру, ліворуч або за шириною.
- **50+ шрифтів, згрупованих за писемністю** — перська, арабська, іврит, урду, латиниця,
  кирилиця, деванагарі, японська, китайська, корейська, тайська, бенгальська — з можливістю
  пошуку, попереднім переглядом у засобі вибору та фільтрами-чипами для кожної мови. Окремі шрифти
  для основного тексту, заголовків і коду — кожен з них вебшрифт для завантаження (Google Fonts).
- **Типографіка** — розмір, насиченість, висота рядка, інтервали між літерами/словами/абзацами
  та обмеження ширини для читання — усе з повзунками в реальному часі.
- **Локальні цифри** — показуйте числа цифрами вашої мови (перська, арабська, урду, пушту, гінді,
  бенгальська, тайська). Заблоковано з поясненням для мов, які вже використовують стандартні
  західні цифри.
- **Математика й код залишаються коректними** — формули KaTeX і блоки коду залишаються
  написаними зліва направо й незмінними.
- **Тематизація Claude** — перемикайте сам Claude між Auto / Світлою / Темною / висококонтрастною
  Чорною темами, а також додавайте власний акцентний колір.
- **Інтерфейс 25 мовами** — панель налаштувань під час першого запуску автоматично визначає мову
  вашого браузера й повністю перекладена 25 мовами.
- **Попередній перегляд у реальному часі** — побачте, як саме виглядатиме Claude (його власна тема
  + акцент) перед застосуванням.
- **Пресети, експорт/імпорт, власний CSS**, а також відшліфований, доступний з клавіатури інтерфейс
  зі світлою / темною / чорною / системною темами панелі.

## 🚀 Встановлення (режим розробника)

1. Відкрийте `chrome://extensions` (Chrome, Edge або Brave).
2. Увімкніть **Режим розробника** (вгорі праворуч).
3. Натисніть **Завантажити розпаковане** та оберіть папку цього проєкту.
4. Відкрийте чат Claude і натисніть значок розширення, щоб відкрити налаштування.

Комбінації клавіш можна призначити на сторінці `chrome://extensions/shortcuts` (за замовчуванням
жодних не задано).

## 🔒 Конфіденційність

Розширення **не збирає жодних даних** — без аналітики, без телеметрії, без відстеження.
Налаштування зберігаються лише в `chrome.storage.local` на вашому пристрої. Єдині мережеві
запити — це **файли шрифтів**, що завантажуються з Google Fonts, коли ви обираєте
вебшрифт для завантаження. Докладніше див. [PRIVACY.md](PRIVACY.md).

## 📁 Структура проєкту

```
.
├── manifest.json              # MV3 manifest
├── icons/                     # icon.svg + 16/32/48/128 PNGs
├── _locales/                  # chrome.i18n manifest strings (25 languages)
├── PRIVACY.md
└── src/
    ├── lib/
    │   ├── defaults.js        # default settings, merge/validation, language + digit maps
    │   ├── fonts.js           # font catalog (grouped by script)
    │   └── i18n.js            # 25-language UI translations
    ├── content/
    │   ├── content.js         # core engine (direction, scope, theme, digits)
    │   └── inject.css         # injected page styles
    ├── background/
    │   └── sw.js              # service worker (re-injection, shortcut)
    └── popup/
        ├── popup.html
        ├── theme-init.js
        ├── popup.css
        └── popup.js
```
