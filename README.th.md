ส่วนขยายเบราว์เซอร์แบบ Manifest V3 ที่ทำให้เว็บแอป **Claude** (`claude.ai` /
`claude.com`) เป็นแบบขวาไปซ้าย และช่วยให้คุณปรับแต่งตัวอักษรและธีมได้อย่างเต็มที่

![icon](icons/icon128.png)

> **ส่วนขยายอิสระที่ไม่เป็นทางการ — ไม่ได้มีส่วนเกี่ยวข้อง ไม่ได้รับการรับรอง และไม่ได้
> รับการสนับสนุนจาก Anthropic “Claude” เป็นเครื่องหมายการค้าของ Anthropic**

## ✨ ฟีเจอร์

- **ขวาไปซ้ายและทิศทาง** — `Auto` (ตรวจจับแยกตามบล็อก เหมาะที่สุดสำหรับ
  แชตสองภาษา), บังคับ RTL หรือบังคับ LTR ใช้งานแยกกันได้สำหรับคำตอบของ Claude
  ข้อความของคุณ ช่องเขียนข้อความ และรายการแชตในแถบด้านข้าง
- **การจัดวาง** — เริ่มต้น (ตามทิศทาง), ขวา, กึ่งกลาง, ซ้าย หรือชิดขอบทั้งสองด้าน
- **ฟอนต์มากกว่า 50 แบบ จัดกลุ่มตามชนิดอักษร** — เปอร์เซีย, อาหรับ, ฮีบรู, อูรดู, ละติน,
  ซีริลลิก, เทวนาครี, ญี่ปุ่น, จีน, เกาหลี, ไทย, เบงกาลี — พร้อมตัวเลือกฟอนต์ที่ค้นหาได้
  แสดงตัวอย่างได้ และมีชิปกรองแยกตามภาษา ตั้งฟอนต์แยกกันได้สำหรับเนื้อความ หัวข้อ
  และโค้ด — แต่ละแบบเป็นเว็บฟอนต์ที่ดาวน์โหลดได้ (Google Fonts)
- **ตัวอักษร** — ขนาดฟอนต์ น้ำหนัก ความสูงบรรทัด ระยะห่างระหว่างตัวอักษร/คำ/ย่อหน้า
  และการจำกัดความกว้างสำหรับการอ่าน ทั้งหมดปรับได้ด้วยแถบเลื่อนแบบเรียลไทม์
- **ตัวเลขท้องถิ่น** — แสดงตัวเลขด้วยเลขโดดของภาษาคุณเอง (เปอร์เซีย,
  อาหรับ, อูรดู, ปาทาน, ฮินดี, เบงกาลี, ไทย) ล็อกไว้พร้อมคำอธิบายสำหรับภาษา
  ที่ใช้ตัวเลขตะวันตกมาตรฐานอยู่แล้ว
- **คณิตและโค้ดยังคงถูกต้อง** — สูตร KaTeX และบล็อกโค้ดยังคงเป็นแบบ
  ซ้ายไปขวาและไม่ผิดเพี้ยน
- **การปรับธีมของ Claude** — สลับ Claude เองระหว่าง Auto / สว่าง / มืด /
  ดำแบบคอนทราสต์สูง พร้อมสีเน้นแบบกำหนดเอง
- **อินเทอร์เฟซ 25 ภาษา** — แผงตั้งค่าจะตรวจจับภาษาเบราว์เซอร์ของคุณ
  อัตโนมัติในการเปิดใช้ครั้งแรก และแปลครบถ้วนเป็น 25 ภาษา
- **ตัวอย่างแบบเรียลไทม์** — ดูได้อย่างชัดเจนว่า Claude จะมีหน้าตาอย่างไร (ธีมและสีเน้นของมันเอง)
  ก่อนนำไปใช้จริง
- **พรีเซ็ต, ส่งออก/นำเข้า, CSS แบบกำหนดเอง** และ UI ที่ขัดเกลามาอย่างดี เข้าถึงได้ด้วย
  แป้นพิมพ์ พร้อมธีมแผงแบบสว่าง / มืด / ดำ / ตามระบบ

## 🚀 ติดตั้ง (โหมดนักพัฒนา)

1. เปิด `chrome://extensions` (Chrome, Edge หรือ Brave)
2. เปิดใช้ **Developer mode** (มุมขวาบน)
3. คลิก **Load unpacked** แล้วเลือกโฟลเดอร์โปรเจกต์นี้
4. เปิดแชต Claude แล้วคลิกไอคอนส่วนขยายเพื่อเปิดการตั้งค่า

สามารถกำหนดทางลัดแป้นพิมพ์ได้ที่ `chrome://extensions/shortcuts` (โดยค่าเริ่มต้น
ไม่ได้ตั้งไว้)

## 🔒 ความเป็นส่วนตัว

ส่วนขยายนี้ **ไม่เก็บข้อมูลใด ๆ** — ไม่มีการวิเคราะห์ ไม่มีการเก็บข้อมูลการใช้งาน ไม่มีการติดตาม
การตั้งค่าจะถูกเก็บไว้เฉพาะใน `chrome.storage.local` บนอุปกรณ์ของคุณ คำขอเครือข่ายอย่างเดียว
ที่มีคือ **ไฟล์ฟอนต์** ที่ดึงจาก Google Fonts เมื่อคุณเลือกฟอนต์ที่ดาวน์โหลดได้
ดูรายละเอียดได้ที่ [PRIVACY.md](PRIVACY.md)
